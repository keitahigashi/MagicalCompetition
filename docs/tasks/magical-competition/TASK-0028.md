## 実装詳細

### 1. AIInfoView 🔵

**信頼性**: 🔵 *architecture.md AIInfoViewより*

AIプレイヤーの情報を表示するコンポーネント。手札枚数・山札枚数・リーチ状態・思考中アニメーションを管理する。

```csharp
public class AIInfoView : MonoBehaviour
{
    [SerializeField] private Text _aiNameText;
    [SerializeField] private Text _handCountText;
    [SerializeField] private Text _deckCountText;
    [SerializeField] private GameObject _reachIcon;    // リーチアイコン
    [SerializeField] private GameObject _thinkingIcon; // 思考中表示

    public void UpdateInfo(PlayerState aiPlayer)
    {
        // AI名、手札枚数、山札枚数を更新
        // リーチ状態の表示切替
    }

    public void ShowThinking()
    {
        // 思考中アイコンを表示
    }

    public void HideThinking()
    {
        // 思考中アイコンを非表示
    }
}
```

- AI名/ID表示
- 手札枚数表示
- 山札枚数表示
- リーチ状態表示（アイコン）（REQ-403）
- 思考中表示のON/OFF（REQ-302）

**実装ファイル**: `Assets/_Project/Scripts/Views/AIInfoView.cs`
