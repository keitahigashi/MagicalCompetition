## 実装詳細

### 1. LoadGameScene(int aiCount) 🔵

**信頼性**: 🔵 *REQ-401 シーン遷移より*

AI人数を保持してGameSceneへ遷移する。

```csharp
public class SceneController
{
    public static int AICount { get; private set; }

    public void LoadGameScene(int aiCount)
    {
        AICount = aiCount;
        SceneManager.LoadScene("GameScene");
    }
}
```

- AI人数（1〜4）をstatic変数に保持
- SceneManager.LoadScene("GameScene") でシーン遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/SceneController.cs`

### 2. LoadTitleScene() 🔵

**信頼性**: 🔵 *architecture.md Controllersより*

TitleSceneへ遷移する。

```csharp
public void LoadTitleScene()
{
    SceneManager.LoadScene("TitleScene");
}
```

- ゲーム終了後やリトライ時にタイトル画面へ戻る

**実装ファイル**: `Assets/_Project/Scripts/Controllers/SceneController.cs`

### 3. シーン間データ受け渡し 🔵

**信頼性**: 🔵 *architecture.md データ受け渡しより*

static変数を使用してシーン間でデータを共有する。

```csharp
public static int AICount { get; private set; }
public static DifficultyLevel Difficulty { get; private set; }
```

- GameScene側ではSceneController.AICountを参照してGameConfigを生成する
- 将来的にScriptableObjectベースに移行可能な設計にする

**実装ファイル**: `Assets/_Project/Scripts/Controllers/SceneController.cs`
