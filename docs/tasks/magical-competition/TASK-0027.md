## タスク概要

場札エリアの表示（FieldView）と山札の残り枚数表示（DeckView）を実装する。FieldViewは現在の場札カードの色・番号を明確に表示し、疑似場札時の特殊表示にも対応する。DeckViewは山札の裏面画像と残り枚数テキスト、リーチ状態の表示を行う。

## 実装詳細

### 1. FieldView 🔵

**信頼性**: 🔵 *architecture.md FieldViewより*

場札カードの表示コンポーネント。CardViewを使用して場札を表示し、色・番号を明確に視認できるようにする。

```csharp
public class FieldView : MonoBehaviour
{
    [SerializeField] private CardView _fieldCardView;
    [SerializeField] private Text _fieldInfoText; // 色・番号テキスト

    public void UpdateField(FieldState field)
    {
        // 場札カードの表示更新
        // 疑似場札時は特殊表示（「好きな色の5」等）
    }
}
```

- 場札カードの色・番号を明確に表示（NFR-104）
- 疑似場札時の表示（「好きな色の5」等の特殊テキスト）
- 場札更新時のアニメーション

**実装ファイル**: `Assets/_Project/Scripts/Views/FieldView.cs`

### 2. DeckView 🔵

**信頼性**: 🔵 *architecture.md DeckViewより*

山札の残り枚数表示コンポーネント。カード裏面画像と残り枚数テキストを表示する。

```csharp
public class DeckView : MonoBehaviour
{
    [SerializeField] private Image _deckImage;     // カード裏面画像
    [SerializeField] private Text _countText;      // 残り枚数テキスト

    public void UpdateCount(int count)
    {
        // 残り枚数テキスト更新
        // 0枚時にリーチ表示
    }
}
```

- 山札の残り枚数表示（REQ-403）
- カード裏面画像 + 残り枚数テキスト
- リーチ状態（残り少数枚）の表示

**実装ファイル**: `Assets/_Project/Scripts/Views/DeckView.cs`
