## 実装詳細

### 1. PlayerState class 🔵

**信頼性**: 🔵 *interfaces.md PlayerStateより*

- **Properties**:
  - `PlayerId` (int) - プレイヤー識別子
  - `IsHuman` (bool) - 人間プレイヤーかどうか
  - `Hand` (List\<Card\>) - 手札
  - `Deck` (List\<Card\>) - 山札
  - `IsReach` (bool) - リーチ状態

- **Computed Properties**:
  - `TotalCardCount` - Hand.Count + Deck.Count
  - `HasWon` - TotalCardCount == 0

- **Methods**:
  - `AddToHand(Card card)` - 手札にカードを追加
  - `RemoveFromHand(Card card)` - 手札から1枚削除
  - `RemoveFromHand(List<Card> cards)` - 手札から複数枚削除
  - `DrawFromDeck()` - 山札先頭から1枚引いて返す
  - `ReturnToDeckBottom(Card card)` - カードを山札の末尾に追加

**実装ファイル**: `Assets/_Project/Scripts/Core/Model/PlayerState.cs`

## 単体テスト要件

### 1. 初期状態テスト 🔵

手札・山札が空、IsReach=falseであることを確認する。

### 2. AddToHandテスト 🔵

カードが手札に正しく追加されることを確認する。

### 3. RemoveFromHand(1枚)テスト 🔵

指定したカードが手札から削除されることを確認する。

### 4. RemoveFromHand(複数枚)テスト 🔵

指定した複数カードが全て手札から削除されることを確認する。

### 5. DrawFromDeckテスト 🔵

山札先頭から1枚引いて返すことを確認する。

### 6. DrawFromDeck(山札空)テスト 🔵

山札が空の場合に例外またはnullが返ることを確認する。

### 7. ReturnToDeckBottomテスト 🔵

カードが山札の末尾に追加されることを確認する。

### 8. TotalCardCountテスト 🔵

手札+山札の合計が正しく計算されることを確認する。

### 9. HasWonテスト 🔵

合計0枚でtrueが返ることを確認する。
