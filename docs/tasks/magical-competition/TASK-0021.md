## 実装詳細

### 1. Drawフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md Drawフローより*

カード出し後の手札補充処理を行う。

```csharp
private void ExecuteDraw()
```

- DrawSystem.Refill(currentPlayer) で手札を3枚まで補充
- 山札が空になった場合、IsReachフラグをtrueに設定
- → CheckWinフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`（TASK-0020からの追加）

### 2. CheckWinフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md CheckWinフローより*

勝利条件の判定を行う。

```csharp
private void ExecuteCheckWin()
```

- WinChecker.Check(currentPlayer) で勝利判定
- 勝利の場合:
  - ScoreCalculator.Calculate() でスコア算出
  - PlayResult生成（Winner, Score, AllPlayers）
  - → Endフェーズへ遷移
- 継続の場合:
  - → NextTurnフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`（TASK-0020からの追加）

### 3. NextTurnフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md NextTurnフローより*

次のプレイヤーへターンを進める。

```csharp
private void ExecuteNextTurn()
```

- TurnManager.AdvanceTurn(state) で次プレイヤーに切り替え
- → Playフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`（TASK-0020からの追加）

### 4. AllPassResetフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md 全員パスリセットフローより*

全員パス時のリセット処理を行う。

```csharp
private void ExecuteAllPassReset()
```

- TurnManager.ResetForNewRound(state) でパスカウントリセット
- FieldUpdater.ClearField(state.Field) で場札をクリア
- → SelectFieldCardフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`（TASK-0020からの追加）

### 5. Endフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md ゲーム終了フローより*

ゲーム終了処理を行う。

```csharp
private void ExecuteEnd()
```

- PlayResultを保持し、外部（UIなど）から参照可能にする
- ResultDialogViewの表示を待つ

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`（TASK-0020からの追加）

## 実装手順

### TDD手順

1. **tdd-requirements**: dataflow.mdからDraw/CheckWin/NextTurn/AllPassReset/Endフェーズのフローを確認する
2. **tdd-testcases**: 上記6つのテストケースをGameControllerDrawTests.csに記述する
3. **tdd-red**: テストを実行し、全てREDであることを確認する
4. **tdd-green**: GameControllerクラスの後半フェーズを実装し、全テストをGREENにする
   - Draw: DrawSystem.Refill呼び出し、山札チェック
   - CheckWin: WinChecker.Check → 勝利ならScoreCalculator.Calculate → End遷移
   - NextTurn: TurnManager.AdvanceTurn → Play遷移
   - AllPassReset: TurnManager.ResetForNewRound → FieldUpdater.ClearField → SelectFieldCard遷移
   - End: PlayResult保持
5. **tdd-refactor**: 必要に応じてリファクタリングする（フェーズディスパッチャの整理、エラーハンドリングの追加など）
6. **tdd-verify-complete**: 全テストがパスし、完了条件を満たすことを確認する
