## タスク概要

45枚の魔石カードの生成、シャッフル、全プレイヤーへの均等配布を行うクラスを実装する。デッキ生成では5色×9番号の全カードを生成し、Fisher-Yatesアルゴリズムでシャッフルした後、指定人数のプレイヤーへ均等に配布する。余りカードは除外カードとして返す。

## 実装詳細

### 1. CreateDeck() 🔵

**信頼性**: 🔵 *interfaces.md、REQ-001 カード構成より*

5色（Fire, Water, Light, Earth, Wind）× 9番号（1〜9）= 45枚のCardオブジェクトを生成し、`List<Card>` として返す。

```csharp
public List<Card> CreateDeck()
```

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/CardDistributor.cs`

### 2. Shuffle(List\<Card\> deck) 🔵

**信頼性**: 🔵 *architecture.md シャッフルアルゴリズムより*

Fisher-Yatesアルゴリズムによるインプレースシャッフル。外部から `System.Random` を注入可能にし、テスト時にシード固定できるようにする。

```csharp
public void Shuffle(List<Card> deck)
public void Shuffle(List<Card> deck, System.Random random)
```

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/CardDistributor.cs`

### 3. Distribute(List\<Card\> deck, int playerCount) 🔵

**信頼性**: 🔵 *REQ-501 カード配布ルールより*

デッキを指定人数で均等に分割し、各プレイヤー分の手札リストと余りカードをタプルまたは専用クラスで返す。

```csharp
public DistributeResult Distribute(List<Card> deck, int playerCount)
// DistributeResult { List<List<Card>> PlayerDecks, List<Card> ExcludedCards }
```

- 45 ÷ 2 = 22枚ずつ + 1枚余り
- 45 ÷ 3 = 15枚ずつ + 0枚余り
- 45 ÷ 4 = 11枚ずつ + 1枚余り
- 45 ÷ 5 = 9枚ずつ + 0枚余り

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/CardDistributor.cs`
