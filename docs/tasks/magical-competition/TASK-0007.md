## 実装詳細

### 1. GameConfig class 🔵

**信頼性**: 🔵 *interfaces.md GameConfigより*

- **Properties**:
  - `AICount` (int) - AI対戦相手の数（1~4）

- **Computed Properties**:
  - `TotalPlayerCount` - AICount + 1

- **Constants**:
  - `TotalCards = 45` - 総カード枚数
  - `HandSize = 3` - 初期手札枚数
  - `InitialFieldNumber = 5` - 疑似場札の初期番号

**実装ファイル**: `Assets/_Project/Scripts/Core/Model/GameConfig.cs`

### 2. GameState class 🔵

**信頼性**: 🔵 *interfaces.md GameStateより*

- **Properties**:
  - `Players` (List\<PlayerState\>) - 全プレイヤーの状態
  - `CurrentPhase` (GamePhase) - 現在のゲームフェーズ
  - `CurrentPlayerIndex` (int) - 現在のプレイヤーインデックス
  - `Field` (FieldState) - 場札の状態
  - `ConsecutivePassCount` (int) - 連続パス回数
  - `DiscardPile` (List\<Card\>) - 捨て札
  - `ExcludedCards` (List\<Card\>) - 除外カード

- **Computed Properties**:
  - `CurrentPlayer` - Players[CurrentPlayerIndex]
  - `PlayerCount` - Players.Count
  - `AllPlayersPassed` - ConsecutivePassCount >= PlayerCount

- **Methods**:
  - `AdvanceTurn()` - 次のプレイヤーに切り替え（循環）
  - `ResetPassCount()` - 連続パスカウントを0にリセット

**実装ファイル**: `Assets/_Project/Scripts/Core/Model/GameState.cs`

## 単体テスト要件

### 1. GameConfig_AICount1_TotalPlayerCount2テスト 🔵

AICount=1のときTotalPlayerCount=2であることを確認する。

### 2. GameConfig_AICount4_TotalPlayerCount5テスト 🔵

AICount=4のときTotalPlayerCount=5であることを確認する。

### 3. GameState_AdvanceTurn_NextPlayerテスト 🔵

AdvanceTurnで次のプレイヤーに切り替わることを確認する。

### 4. GameState_AdvanceTurn_WrapsAroundテスト 🔵

最後のプレイヤーの次は先頭（インデックス0）に戻ることを確認する。

### 5. GameState_AllPlayersPassed_TrueWhenEnoughPassesテスト 🔵

連続パス数がプレイヤー数以上のときAllPlayersPassedがtrueであることを確認する。

### 6. GameState_ResetPassCount_ResetsToZeroテスト 🔵

ResetPassCountで連続パスカウントが0にリセットされることを確認する。

**テストファイル**: `Assets/_Project/Tests/EditMode/Core/Model/GameStateTests.cs`, `Assets/_Project/Tests/EditMode/Core/Model/GameConfigTests.cs`
