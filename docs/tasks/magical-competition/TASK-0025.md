## 実装詳細

### 1. カード表示 🔵

**信頼性**: 🔵 *interfaces.md CardViewより*

UI Imageをベースとしたカード表示コンポーネント。CardDataを受け取り、カード画像を表示する。

```csharp
public class CardView : MonoBehaviour
{
    private Image _cardImage;        // UI Image
    public Card CardData { get; private set; }

    public void SetCard(Card card, Sprite sprite)
    {
        CardData = card;
        _cardImage.sprite = sprite;
    }
}
```

**実装ファイル**: `Assets/_Project/Scripts/Views/CardView.cs`
**Prefab**: `Assets/_Project/Prefabs/Cards/CardView.prefab`

### 2. 選択状態 🔵

**信頼性**: 🔵 *interfaces.md CardViewより*

選択時にカードが少し上に浮く視覚フィードバックを提供する。

```csharp
public void SetSelected(bool selected)
{
    // 選択時: RectTransform.anchoredPosition.y += 20
    // 非選択時: 元の位置に戻す
}
```

- 選択状態はboolフラグで管理
- RectTransform.anchoredPosition.yを20px上に移動

### 3. 操作可否 🔵

**信頼性**: 🔵 *NFR-102より*

出せないカードをグレーアウト表示にして、視覚的に操作可否を区別する。

```csharp
public void SetInteractable(bool enabled)
{
    // enabled=false: グレーアウト（CanvasGroup.alpha or Image.color）
    // enabled=true: 通常表示に戻す
}
```

- CanvasGroup.alpha または Image.colorで半透明化
- 操作不可時はクリックイベントも無効化

### 4. アニメーション 🔵

**信頼性**: 🔵 *design-interview.md Q3より*

カードの移動・フェードアニメーション機能。DOTween or Coroutineで実装。

```csharp
public void PlayMoveAnimation(Vector3 target, float duration)
{
    // 現在位置からtargetへの移動アニメーション
}

public void PlayFadeAnimation(float targetAlpha, float duration)
{
    // 現在のalphaからtargetAlphaへのフェードアニメーション
}
```

## PlayModeテスト要件

### 1. SetCard: 正しいスプライトが表示される 🔵

SetCardでカードとスプライトを設定した際、UI Imageのspriteが正しく設定されることを確認する。

### 2. SetSelected(true): カードが上に浮く 🔵

SetSelected(true)を呼び出した際、RectTransform.anchoredPosition.yが元の位置より20px上に移動することを確認する。

### 3. SetSelected(false): カードが元の位置に戻る 🔵

SetSelected(true)の後にSetSelected(false)を呼び出した際、RectTransform.anchoredPosition.yが元の位置に戻ることを確認する。

### 4. SetInteractable(false): グレーアウト表示になる 🔵

SetInteractable(false)を呼び出した際、カードがグレーアウト表示（alpha低下 or 色変更）になることを確認する。

### 5. SetInteractable(true): 通常表示に戻る 🔵

SetInteractable(false)の後にSetInteractable(true)を呼び出した際、カードが通常表示に戻ることを確認する。

**テストファイル**: `Assets/_Project/Tests/PlayMode/Views/CardViewTests.cs`
