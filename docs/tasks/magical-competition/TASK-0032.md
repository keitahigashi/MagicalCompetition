## 実装詳細

### 1. Canvas設定 🔵

**信頼性**: 🔵 *REQ-601より*

GameSceneのCanvas基本設定。

- Render Mode: Screen Space - Overlay
- Canvas Scaler: Scale With Screen Size
- Reference Resolution: 1920 x 1080
- Screen Match Mode: Match Width Or Height (Match = 0.5)

**Scene**: `Assets/_Project/Scenes/GameScene.unity`

### 2. レイアウト配置 🔵

**信頼性**: 🔵 *interfaces.md 画面レイアウト定義, design-interview.md Q4より*

Canvas上にUIコンポーネントを以下のレイアウトで配置する。

```
┌───────────────────────────────────────────┐
│          AI情報エリア（上部）               │
│  [AI1: 手札3 山札8] [AI2: 手札2 山札5 ⚡]  │
├───────────────────────────────────────────┤
│          場札エリア（中央）                 │
│        ┌─────────┐                        │
│        │  火 ・ 5 │                        │
│        └─────────┘                        │
│     ターン表示: 「あなたのターン」           │
├───────────────────────────────────────────┤
│  プレイヤーエリア（下部）                   │
│  手札（横並び）    山札                     │
│      [出す]  [パス]                        │
└───────────────────────────────────────────┘
```

配置するコンポーネント:
- **上部**: AIInfoView x N（AI人数分、HorizontalLayoutGroup）
- **中央上**: FieldView（場札表示）
- **中央下**: TurnIndicatorView（ターン表示）
- **下部左**: HandView（プレイヤー手札、横並び）
- **下部右**: DeckView（プレイヤー山札）
- **下部中央**: GameUI（「出す」「パス」ボタン）
- **オーバーレイ**: ResultDialogView（非表示状態で配置）

### 3. コンポーネント配線 🔵

**信頼性**: 🔵 *architecture.md Controllers, Viewsより*

各Controller/View間のSerializeFieldの参照を接続する。

- **GameController**:
  - 各System（PlayValidator, JudgeSystem等）の参照
  - 各View（FieldView, HandView, DeckView等）の参照
  - 各Controller（PlayerInputController, AIController等）の参照
- **PlayerInputController**:
  - HandView の参照
  - GameUI の参照
- **AIController**:
  - AIInfoView の参照
- **全View**:
  - 内部UIコンポーネント（Text, Image, Button等）の参照

## 実装手順

### DIRECT手順

1. **direct-setup**: GameScene.unityを開く（または新規作成する）
2. **direct-setup**: Canvas GameObjectを作成し、Canvas Scaler設定を行う（Screen Space - Overlay、1920x1080基準）
3. **direct-setup**: AI情報エリア（上部）にAIInfoViewプレハブを配置する
4. **direct-setup**: 場札エリア（中央）にFieldViewコンポーネントを配置する
5. **direct-setup**: ターン表示にTurnIndicatorViewコンポーネントを配置する
6. **direct-setup**: プレイヤーエリア（下部）にHandView、DeckView、GameUIを配置する
7. **direct-setup**: ResultDialogViewプレハブをオーバーレイとして配置する（初期非表示）
8. **direct-setup**: GameController GameObjectを作成し、各System/View/Controllerの参照をアタッチする
9. **direct-setup**: PlayerInputController GameObjectを作成し、HandView/GameUIの参照をアタッチする
10. **direct-verify**: Unityエディタで再生し、コンパイルエラー・NullReferenceエラーがないことを確認する
11. **direct-verify**: 各View/UIが画面に正しく表示されることを目視確認する
12. **direct-verify**: レイアウトが1920x1080基準で正しく配置されていることを確認する
