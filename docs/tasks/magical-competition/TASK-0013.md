## 実装詳細

### 1. Update(FieldState field, Card lastPlayedCard) 🔵

**信頼性**: 🔵 *dataflow.md FieldUpdater、REQ-105より*

カードプレイ後に場札を更新する。最後に出したカードの色と番号で場札を上書きする。

```csharp
public void Update(FieldState field, Card lastPlayedCard)
```

- `field.Color = lastPlayedCard.Color`
- `field.Number = lastPlayedCard.Number`
- `field.IsVirtual = false`（通常の場札に戻す）

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/FieldUpdater.cs`

### 2. SetVirtualField(FieldState field, CardColor color) 🔵

**信頼性**: 🔵 *REQ-005, REQ-107 疑似場札ルールより*

全員パス時に設定される疑似場札。指定した色で番号5の仮想場札を設定する。

```csharp
public void SetVirtualField(FieldState field, CardColor color)
```

- `field.Color = color`
- `field.Number = 5`
- `field.IsVirtual = true`

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/FieldUpdater.cs`

### 3. ClearField(FieldState field) 🔵

**信頼性**: 🔵 *REQ-107 場札リセットルールより*

場札をクリアして初期状態に戻す。

```csharp
public void ClearField(FieldState field)
```

- 場札の色・番号をデフォルト値にリセット
- `field.IsVirtual = false`

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/FieldUpdater.cs`

## 単体テスト要件

### 1. Update: 場札がlastPlayedCardの色・番号に更新される 🔵

通常のカードプレイ後に場札が正しく更新されることを確認する。

### 2. Update(同数字複数枚後): 最後のカードの色に更新 🔵 (EDGE-301)

同数字で複数枚出した場合、最後のカードの色で場札が更新されることを確認する。番号は全て同じなのでどのカードでも同じ。

### 3. Update(同色複数枚後): 最後のカードの番号に更新 🔵 (EDGE-302)

同色で複数枚出した場合、最後のカードの番号で場札が更新されることを確認する。色は全て同じなのでどのカードでも同じ。

### 4. Update(計算出し後): 最後のカードの色・番号に更新(計算結果ではない) 🔵 (EDGE-303)

計算出しの場合、計算結果ではなく最後に出したカードの色・番号で場札が更新されることを確認する。

### 5. SetVirtualField: 指定色・番号5・IsVirtual=trueに設定 🔵

疑似場札が指定した色、番号5、IsVirtual=trueで正しく設定されることを確認する。

### 6. ClearField: 場札がクリアされる 🔵

場札が初期状態にリセットされることを確認する。

**テストファイル**: `Assets/_Project/Tests/EditMode/Core/Systems/FieldUpdaterTests.cs`
