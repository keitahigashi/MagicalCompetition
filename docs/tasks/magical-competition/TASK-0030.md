## PlayModeテスト要件

### 1. AI人数選択: 1~4の選択が可能 🔵

各AI人数ボタン（1~4）をクリックした際、SelectedAICountが対応する値に更新されることを確認する。

### 2. 開始ボタン: クリックでOnGameStartイベント発火 🔵

ゲーム開始ボタンをクリックした際、OnGameStartイベントが発火することを確認する。

### 3. 開始ボタン: 選択されたAI人数が正しく取得される 🔵

AI人数を選択してから開始ボタンをクリックした際、OnGameStartイベントに選択されたAI人数が正しく渡されることを確認する。

### 4. 初期状態: デフォルトAI人数が設定されている 🔵

TitleUI初期化時に、デフォルトのAI人数（1体）が設定されていることを確認する。

**テストファイル**: `Assets/_Project/Tests/PlayMode/UI/TitleUITests.cs`
