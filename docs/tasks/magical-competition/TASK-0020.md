## 実装詳細

### 1. Setupフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md セットアップフローより*

ゲーム開始時のセットアップ処理を行う。

```csharp
public class GameController
{
    private void ExecuteSetup()
}
```

- CardDistributor.CreateDeck() でデッキを生成
- CardDistributor.Shuffle() でシャッフル
- CardDistributor.Distribute() で全プレイヤーへ配布
- DrawSystem.InitialDraw(players) で各プレイヤーの初期手札を設定
- FieldUpdater.SetVirtualField(CardColor.Any, 5) で疑似場札を設定
- 余りカードをGameState.ExcludedCardsに保存
- → SelectFieldCardフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`

### 2. SelectFieldCardフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md セットアップフローより*

場札の色選択を行う。

```csharp
private void ExecuteSelectFieldCard()
```

- CurrentPlayerがHumanの場合: PlayerInputControllerに色選択を委譲
- CurrentPlayerがAIの場合: IAIStrategy.SelectFieldColor()で色を自動選択
- FieldUpdater.SetVirtualField(選択された色, 5) で場札を更新
- → Playフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`

### 3. Playフェーズ処理 🔵

**信頼性**: 🔵 *dataflow.md プレイヤーターンより*

プレイヤーのターン制御を行う。

```csharp
private void ExecutePlay()
```

- CurrentPlayerがHumanの場合: PlayerInputControllerに制御を委譲
- CurrentPlayerがAIの場合: AIControllerに制御を委譲
- カード出しの場合:
  - 手札からカードを削除
  - 場札を更新（FieldUpdater）
  - → Drawフェーズへ遷移
- パスの場合:
  - TurnManager.RecordPass() でパスカウント増加
  - 全員パスの場合 → AllPassResetフェーズへ遷移
  - それ以外 → NextTurnフェーズへ遷移

**実装ファイル**: `Assets/_Project/Scripts/Controllers/GameController.cs`

## 単体テスト要件

### 1. Setup: デッキ生成・配布・初期ドロー・場札設定が実行される 🔵

Setupフェーズ実行後に、全プレイヤーに手札が配布され、場札が初期設定されていることを確認する。

### 2. Setup: 余りカードがExcludedCardsに入る 🔵

配布後の余りカードがGameState.ExcludedCardsに正しく保存されていることを確認する。

### 3. SelectFieldCard: 色選択後にPlayフェーズに遷移 🔵

色選択が完了した後にGameStateMachineがPlayフェーズに遷移していることを確認する。

### 4. Play(カード出し): 手札から削除・場札更新・Drawへ遷移 🔵

カード出し時に手札からカードが削除され、場札が更新され、Drawフェーズへ遷移することを確認する。

### 5. Play(パス): パスカウント増加・NextTurnへ遷移 🔵

パス時に連続パスカウントが増加し、全員パスでない場合にNextTurnフェーズへ遷移することを確認する。

### 6. Play(全員パス): AllPassResetへ遷移 🔵

全プレイヤーがパスした場合にAllPassResetフェーズへ遷移することを確認する。

**テストファイル**: `Assets/_Project/Tests/EditMode/Controllers/GameControllerSetupTests.cs`
