## 実装詳細

### 1. Import Settings一括設定 🔵

**信頼性**: 🔵 *tech-stack.md、architecture.md 技術的制約より*

`Assets/_Project/Art/card/` 配下の全JPGファイルに以下のImport Settingsを適用する:

| 項目 | 値 |
|---|---|
| Texture Type | Sprite (2D and UI) |
| Pixels Per Unit | 2000 |
| Filter Mode | Bilinear |
| Max Size | 1024 (WebGL最適化) |
| Compression | Compressed |

**実装ファイル**: `Assets/_Project/Art/card/*.jpg` のImport Settings

### 2. Sprite Atlas作成 🔵

**信頼性**: 🔵 *architecture.md パフォーマンスより*

魔石カード45枚 + 裏面1枚をまとめたSprite Atlasを作成する。

**実装ファイル**: `Assets/_Project/Sprites/Cards/CardAtlas.spriteatlas`

### 3. スプライト参照確認 🔵

**信頼性**: 🔵 *note.md カード素材マッピングより*

- 全45枚の魔石カードが正しくインポートされていることを確認する
- 裏面スプライト(002)が正しくインポートされていることを確認する

**実装ファイル**: `Assets/_Project/Art/card/` 配下の全スプライト

## 実装手順

### DIRECT手順

1. **direct-setup**: `Assets/_Project/Art/card/` 配下の全JPGファイルのImport Settingsを統一する
2. **direct-setup**: TextureImporterを使用したエディタスクリプト、または手動で一括設定する
3. **direct-setup**: Sprite Atlasを作成し、全カードスプライトを登録する
4. **direct-verify**: 全46枚のスプライトがInspectorで正しく表示されることを確認する
5. **direct-verify**: Sprite Atlasのパッキングプレビューでテクスチャがまとまっていることを確認する
