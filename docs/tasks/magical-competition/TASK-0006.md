## 実装詳細

### 1. FieldState class 🔵

**信頼性**: 🔵 *interfaces.md FieldStateより*

- **Properties**:
  - `Color` (CardColor) - 場札の色
  - `Number` (int) - 場札の番号
  - `IsVirtual` (bool) - 疑似場札かどうか

- **Methods**:
  - `Update(Card lastPlayedCard)` - 最後に出したカードの色・番号で更新、IsVirtual=false
  - `SetVirtual(CardColor color)` - 指定色で番号5の疑似場札設定、IsVirtual=true
  - `Clear()` - 色・番号リセット

**実装ファイル**: `Assets/_Project/Scripts/Core/Model/FieldState.cs`
