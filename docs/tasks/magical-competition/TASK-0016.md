## 実装詳細

### 1. AdvanceTurn(GameState state) 🔵

**信頼性**: 🔵 *dataflow.md NextTurn フェーズより*

次のプレイヤーに切り替える。GameStateのAdvanceTurnメソッドを呼び出す。

```csharp
public void AdvanceTurn(GameState state)
```

- `state.CurrentPlayerIndex = (state.CurrentPlayerIndex + 1) % state.Players.Count`
- GameState.AdvanceTurnが存在すればそれに委譲、なければ直接インデックスを操作

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/TurnManager.cs`

### 2. RecordPass(GameState state) 🔵

**信頼性**: 🔵 *REQ-106 パス記録ルールより*

パスを記録し、連続パスカウントを増加させる。

```csharp
public void RecordPass(GameState state)
```

- `state.ConsecutivePassCount++`

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/TurnManager.cs`

### 3. RecordPlay(GameState state) 🔵

**信頼性**: 🔵 *REQ-106 パスカウントリセットルールより*

カードプレイ時に連続パスカウントをリセットする。

```csharp
public void RecordPlay(GameState state)
```

- `state.ConsecutivePassCount = 0`

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/TurnManager.cs`

### 4. IsAllPlayersPassed(GameState state) 🔵

**信頼性**: 🔵 *REQ-107 全員パス判定ルールより*

連続パスカウントがプレイヤー数以上かどうかで全員パスを判定する。

```csharp
public bool IsAllPlayersPassed(GameState state)
```

- `state.ConsecutivePassCount >= state.Players.Count`

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/TurnManager.cs`

### 5. ResetForNewRound(GameState state) 🔵

**信頼性**: 🔵 *REQ-107 ラウンドリセットルールより*

新ラウンド開始時にパスカウントをリセットする。

```csharp
public void ResetForNewRound(GameState state)
```

- `state.ConsecutivePassCount = 0`

**実装ファイル**: `Assets/_Project/Scripts/Core/Systems/TurnManager.cs`

## 単体テスト要件

### 1. AdvanceTurn: 次のプレイヤーインデックスに進む 🔵

CurrentPlayerIndexが0から1に進むことを確認する。

### 2. AdvanceTurn: 最後→先頭に巻き戻る 🔵

最後のプレイヤー（例: 2人の場合インデックス1）からAdvanceTurnで先頭（インデックス0）に巻き戻ることを確認する。

### 3. RecordPass: パスカウントが増える 🔵

RecordPass実行後にConsecutivePassCountが1増加することを確認する。

### 4. RecordPlay: パスカウントが0にリセット 🔵

パスカウントが2の状態からRecordPlayを実行し、ConsecutivePassCountが0にリセットされることを確認する。

### 5. IsAllPlayersPassed(2人,2パス): true 🔵

2人プレイでConsecutivePassCountが2の場合にtrueが返されることを確認する。

### 6. IsAllPlayersPassed(3人,2パス): false 🔵

3人プレイでConsecutivePassCountが2の場合にfalseが返されることを確認する。

### 7. IsAllPlayersPassed(5人,5パス): true 🔵

5人プレイでConsecutivePassCountが5の場合にtrueが返されることを確認する。

**テストファイル**: `Assets/_Project/Tests/EditMode/Core/Systems/TurnManagerTests.cs`
