## 実装詳細

### 1. ExecuteTurn(GameState state, IAIStrategy strategy) 🔵

**信頼性**: 🔵 *dataflow.md AIターンフローより*

AIのターン全体を実行する。

```csharp
public class AIController
{
    public async Task<PlayAction> ExecuteTurn(GameState state, IAIStrategy strategy)
}
```

処理フロー:
1. AIInfoView.ShowThinking() で思考中表示をON
2. AIActionEvaluator.FindAllValidPlays() で全有効手を探索
3. strategy.DecideAction(state, validPlays) で最適手を選択
4. 思考遅延（自然な見た目のため、合計500ms以内）(NFR-003)
5. アクション実行（カードプレイ or パス）
6. AIInfoView.HideThinking() で思考中表示をOFF
7. 選択されたPlayActionを返す

**実装ファイル**: `Assets/_Project/Scripts/Controllers/AIController.cs`

### 2. 非同期処理 🔵

**信頼性**: 🔵 *architecture.md 非同期処理方針より*

async/awaitを使用して思考遅延を制御する。

```csharp
private async Task SimulateThinkingDelay()
```

- 最小遅延: 200ms（即座に決まると不自然）
- 最大遅延: 500ms（NFR-003準拠）
- 実際の思考時間を差し引いて残り時間を待つ

**実装ファイル**: `Assets/_Project/Scripts/Controllers/AIController.cs`

## 実装手順

### TDD手順

1. **tdd-requirements**: dataflow.mdからAIターンフローの仕様、NFR-003の思考時間制約を確認する
2. **tdd-testcases**: 上記4つのテストケースをAIControllerTests.csに記述する
3. **tdd-red**: テストを実行し、全てREDであることを確認する
4. **tdd-green**: AIControllerクラスを実装し、全テストをGREENにする
   - ExecuteTurn: AIActionEvaluator → IAIStrategy.DecideAction の順で呼び出し
   - 思考遅延: Stopwatchで実際の思考時間を計測し、残り時間をawait
   - パス処理: 有効手が空の場合はPlayAction.CreatePass()を返す
   - 思考中フラグ: IsThinkingプロパティで管理
5. **tdd-refactor**: 必要に応じてリファクタリングする（遅延制御の設定外部化、テスト用のTime抽象化など）
6. **tdd-verify-complete**: 全テストがパスし、完了条件を満たすことを確認する
