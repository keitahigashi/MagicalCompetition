## 実装詳細

### 1. PlayAction class 🔵

**信頼性**: 🔵 *interfaces.md PlayActionより*

- **Properties**:
  - `Type` (PlayType) - アクション種別
  - `Cards` (List\<Card\>) - 出すカードのリスト

- **Computed Properties**:
  - `LastCard` - Cards.LastOrDefault()（最後に出したカード）

- **Factory Methods**:
  - `CreatePlay(PlayType type, List<Card> cards)` - カード出しアクション生成
  - `CreatePass(List<Card> cardsToReturn)` - パスアクション生成（TypeはPass、戻すカードを指定）

**実装ファイル**: `Assets/_Project/Scripts/Core/Model/PlayAction.cs`

### 2. PlayResult class 🔵

**信頼性**: 🔵 *interfaces.md PlayResultより*

- **Properties**:
  - `Winner` (PlayerState) - 勝者
  - `Score` (int) - 勝者のスコア
  - `AllPlayers` (List\<PlayerState\>) - 全プレイヤーの最終状態

**実装ファイル**: `Assets/_Project/Scripts/Core/Model/PlayResult.cs`

## 単体テスト要件

### 1. PlayAction_CreatePlay_CorrectTypeAndCardsテスト 🔵

CreatePlayで正しいTypeとCardsが設定されることを確認する。

### 2. PlayAction_CreatePass_TypeIsPassテスト 🔵

CreatePassでTypeがPassとなり、戻すカードが設定されることを確認する。

### 3. PlayAction_LastCard_ReturnsLastCardテスト 🔵

複数カードがある場合にLastCardが最後のカードを返すことを確認する。

### 4. PlayAction_LastCard_EmptyList_ReturnsNullテスト 🔵

カードリストが空の場合にLastCardがnullを返すことを確認する。

### 5. PlayResult_CorrectPropertiesテスト 🔵

Winner, Score, AllPlayersが正しく設定されることを確認する。

**テストファイル**: `Assets/_Project/Tests/EditMode/Core/Model/PlayActionTests.cs`, `Assets/_Project/Tests/EditMode/Core/Model/PlayResultTests.cs`
