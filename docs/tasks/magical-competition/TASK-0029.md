## 実装詳細

### 1. ResultDialogView 🔵

**信頼性**: 🔵 *architecture.md ResultDialogViewより*

ゲーム結果を表示するモーダルダイアログコンポーネント。

```csharp
public class ResultDialogView : MonoBehaviour
{
    [SerializeField] private Text _winnerText;
    [SerializeField] private Text _scoreText;
    [SerializeField] private Transform _playerResultContainer;
    [SerializeField] private Button _titleButton;
    [SerializeField] private GameObject _dialogPanel;

    public event Action OnTitleButtonClicked;

    public void Show(PlayResult result)
    {
        // 勝者表示（プレイヤー or AI名）
        // 得点表示（他PLの残カード合計枚数）(REQ-202)
        // 各プレイヤーの残カード枚数リスト
        // ダイアログを表示
    }

    public void Hide()
    {
        // ダイアログを非表示
    }
}
```

- 勝者表示（プレイヤー or AI名）
- 得点表示（他PLの残カード合計枚数）（REQ-202）
- 各プレイヤーの残カード枚数リスト
- 「タイトルに戻る」ボタン（REQ-404）
- OnTitleButtonClicked event

**実装ファイル**: `Assets/_Project/Scripts/Views/ResultDialogView.cs`
**Prefab**: `Assets/_Project/Prefabs/UI/ResultDialog.prefab`
