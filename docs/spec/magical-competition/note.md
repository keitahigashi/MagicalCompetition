# Magical Competition コンテキストノート

**作成日**: 2026-03-13

## 技術スタック

- **エンジン**: Unity 6.3 (6000.1.x)
- **言語**: C# 9.0+ / .NET Standard 2.1
- **レンダリング**: Built-in Render Pipeline
- **UI**: uGUI (Canvas) / Screen Space - Overlay
- **アーキテクチャ**: MVC + State Machine + ScriptableObject
- **ビルドターゲット**: WebGL
- **テスト**: Unity Test Framework (NUnit 3.x)

## 開発ルール

- 依存は一方向（循環参照禁止）
- Core/Systems のクラスは必ず EditMode テストを書く
- 1フェーズごとに動作確認してから次へ
- `TODO:` コメントで未確定箇所を明示
